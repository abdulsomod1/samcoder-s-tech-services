# Templates Directory

This directory contains templates for various services offered by SAMCODERS WORLD.

## Structure

- `web-development/`: Templates related to web development projects
- `app-development/`: Templates for mobile and web app development
- `graphics-design/`: Design templates and assets
- `bot-referrals/`: Templates for bot referral and growth services
- `review-services/`: Templates for Play Store and App Store review services
- `selling-members/`: Templates for selling active group members
- `cac-registration/`: Templates for Corporate Affairs Commission registration services

## Usage

Place relevant template files (HTML, CSS, images, documents, etc.) in the appropriate subfolder. Each service folder can contain multiple templates or variations.

## Adding New Templates

1. Identify the appropriate service folder
2. Add your template files
3. Update this README if necessary
